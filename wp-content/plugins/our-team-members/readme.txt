Our Team Members WordPress Plugin can show a nice team members grid with their names, photos, bios, titles, abilities, social media icons, and more.

== Description ==

Our Team Members WordPress Plugin helps you to show the team members in a couple of different style with different contents. It allows you to add custom social icons to each team member. It has a couple of different skins for different types of design. A powerful shortcode builder allows you to create the team member shortcode shortly and uses it anywhere on your site to show the team members.

Our Team Members WordPress Plugin has Visual Composer, King Composer, and Elementor page builder support. If you are using any of the builders, you can use our builder element to show the team members.

It has category options. You can add different team members in different categories. Then you can create multiple team shortcodes to show multiple team member grids.

== Installation ==

* Install it as a regular WordPress plugin

* Use this pluginâ€™s ShortCode to show the team member on your site.

* You can adjust the plugin from the plugin settings and shortcode parameters.

Method 1 (Upload):

1. Go to your Wordpress admin then go to Plugins > Add new > Upload

2. Then select the installable file fom you download folder.

3. Upload and active the plugin and you are ready to go.

Method 2 (FTP):

1. Upload the plugin to your server plugin directory and make sure you don't upload the zip file.

2. Then go back to your Wordpress admin and go to Plugins > Installed Plugins

3. If you successfully upload the plugin You will find Our Team Members here.

4. Active the plugin and you are ready to go.
